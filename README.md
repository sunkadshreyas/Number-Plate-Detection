# Number-Plate-Detection
The Java project is for Vehicle Number Plate Detection.
Adding images of Number plates it displays the text.

GUI consists of three buttons and ImageFrame and TextArea.
With the first button we can add images fromm file present locally on the system.
With the second button we invoke the OCR function to convert to text.
With the third button the text is displayed in the TextArea.

Tools used :
1) Tesseract OCR Engine for reading the text from image.
2) JavaFX for GUI

For running the application firstly install Tesseract into the local system.
Change the path names in Main Class.
Provide images which are not  blurred.
